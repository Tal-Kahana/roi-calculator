# 📊 מחשבון ROI לשיווק דיגיטלי

כלי אינטרנטי פשוט לחישוב תשואה על השקעה שיווקית — עם גרף, כרטיסי תוצאות והמלצת AI.

---

## מה הפרויקט עושה

מאפשר לבעלי עסקים ומשווקים להזין שלושה נתונים:

- **הכנסות חודשיות נוכחיות** — מצב ההכנסות היום
- **תקציב שיווק חודשי** — כמה משקיעים בשיווק
- **אחוז צמיחה שנתי משוער** — הצמיחה הצפויה כתוצאה מהשיווק

ומחשב:

| תוצאה | הסבר |
|---|---|
| הכנסות חזויות ל-3 / 6 / 12 חודשים | לפי נוסחת ריבית דריבית שנתית |
| הכנסות שנתיות חזויות | הכנסה חודשית בחודש ה-12 × 12 |
| תקציב שיווק שנתי | תקציב חודשי × 12 |
| רווח / הפסד כולל | הכנסות חזויות − הכנסות נוכחיות − תקציב שיווק |
| ROI שיווקי | ((רווח כולל) / תקציב שנתי) × 100 |

בנוסף מציג גרף קווי של עליית ההכנסות לאורך 12 חודשים, ואפשרות לקבל המלצה מ-GPT.

## איך מריצים

הפרויקט הוא קובץ HTML בודד — אין צורך בהתקנות.

**הרצה מקומית:**
```bash
# פשוט פתחו את הקובץ בדפדפן
open index.html
```

**הרצה עם שרת מקומי (לצפייה בסלולרי):**
```bash
# אם יש Python
python -m http.server 8080

# אם יש Node.js
npx serve .
```

ואז פתחו בדפדפן: `http://localhost:8080`

**GitHub Pages:**
הפרויקט זמין בכתובת:
```
https://YOUR_USERNAME.github.io/roi-calculator
```

## חיבור ל-AI

הכפתור "קבל המלצה מ-AI" שולח את נתוני החישוב ל-GPT-4o-mini של OpenAI ומבקש ניתוח בעברית.

**הגדרה:**

1. קבלו API Key מ-[platform.openai.com/api-keys](https://platform.openai.com/api-keys)
2. פתחו את `index.html` ובשורה הראשונה ב-`<script>` החליפו:

```js
const OPENAI_API_KEY = "החליפו כאן את ה-API Key שלכם";
```

ב:

```js
const OPENAI_API_KEY = "sk-proj-...";
```

> ⚠️ אל תעלו את ה-API Key ל-GitHub — זה מפתח פרטי שעלול לגרור עלויות אם ייחשף.

## מה למדתי

הפרויקט הזה נבנה בשיחה אחת עם **Claude Code**, ולמדתי דרכו מספר דברים:

- **חישובי ROI** — ההבדל בין צמיחה חודשית לשנתית, ואיך ממירים ביניהן עם נוסחת `(1 + שנתי)^(N/12)`
- **Chart.js** — איך בונים גרף קווי עם שני datasets, tooltip מותאם, וצירים מעוצבים
- **עיצוב רספונסיבי** — שימוש ב-CSS Grid עם `repeat(3, 1fr)` ו-media queries למובייל
- **אנימציות CSS** — `@keyframes fadeInUp` עם stagger delays לתחושת כניסה הדרגתית
- **fetch API** — שליחת בקשת POST ל-OpenAI עם headers, error handling ו-async/await
- **GitHub Pages** — העלאת פרויקט סטטי לאוויר בחינם
